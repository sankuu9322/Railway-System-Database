#  Railway System Database (MySQL)

##  Project Overview
This project is a Railway System Database designed using MySQL.  
It manages railway operations including trains, stations, routes, schedules, passengers, ticket bookings, and payments.

## Technologies Used
- MySQL
- MySQL Workbench
- SQL

##  Database Tables
- stations
- trains
- routes
- train_schedule
- passengers
- tickets
- payments

##  Features
- Train and station management
- Route and schedule tracking
- Passenger ticket booking system
- Payment management
- Use of primary and foreign keys

##  How to Run
1. Open MySQL Workbench
2. Import `railway_system.sql`
3. Execute the script
4. Run queries from `queries.sql`

##  Use Case
This project demonstrates a real-world railway reservation system using relational database concepts.
